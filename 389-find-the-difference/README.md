# 389 - Find the Difference

**Difficulty:** Easy  
**Problem Link:** [find-the-difference](https://leetcode.com/problems/find-the-difference/)

## Problem Description

You are given two strings `s` and `t`. String `t` is generated by random shuffling string `s` and then adding one more letter at a random position. Return the letter that was added to `t`.

## Solution Approach

**XOR Approach:**
1. XOR all characters from both strings
2. XOR cancels out matching characters
3. Remaining value is the added character

## Complexity Analysis

- **Time Complexity:** O(n) where n = length of t
- **Space Complexity:** O(1)

## Best Code Example

```javascript
function findTheDifference(s, t) {
    let xor = 0;

    for (let ch of s + t) {
        xor ^= ch.charCodeAt(0);
    }

    return String.fromCharCode(xor);
}
```
